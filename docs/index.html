じぱんぐ学園へようこそ！

ようこそ、otaku-kun。
ここは世界随一の「ハッカー養成学校」として名を馳せた、国家が支援する「じぱんぐ学園」。
しかし今ではかつてのエリート校としての面影はなく、ほとんどの生徒が劣等感を感じている・・・

それでも！私はこの学園を卒業する前にもう一度強豪校にしたい・・・！

学園対抗のCTFやプログラミングコンテストで勝ち上がり、じぱんぐ学園の栄光を取り戻すために、一緒にハッカーとしてのスキルを高めるぞ！
